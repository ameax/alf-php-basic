# Base PHP library from Alf.

Base PHP library from Alf4712.

## Installation

You can install the package via composer:

```bash
composer require alf/alf-php-basic
```

## Usage

```php
use Alf\AlfPhp\types\scalars\AlfInt;
$obj = new AlfInt();
print_r($obj);
```

## Testing

```bash
composer test
```
